
# 📝 Full Stack To-Do List App

A simple and easy-to-understand CRUD To-Do List built with **ReactJS**, **Node.js**, **Express**, and **MongoDB**.



## 🔧 Tech Stack

- **Frontend**: ReactJS
- **Backend**: Node.js + Express
- **Database**: MongoDB (local or MongoDB Atlas)
- **Tooling**: Axios, CORS
